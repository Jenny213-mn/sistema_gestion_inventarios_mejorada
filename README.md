# Sistema de Gestión de Inventarios 

## Descripción
Este proyecto es un sistema de gestión de inventarios desarrollado en Python. 
Permite añadir, actualizar y eliminar productos, almacenando la información en un archivo de texto (`inventario.txt`).

## Características
- Añadir productos
- Actualizar productos
- Eliminar productos
- Mostrar inventario
- Almacenamiento persistente en archivo
- Manejo de excepciones (FileNotFoundError, PermissionError, etc.)
- Creación automática del archivo si no existe

## Tecnologías utilizadas
- Python 
- Manejo de archivos
- Programación orientada a objetos
- Manejo de excepciones
